# rna-seq-analysis-arabidopsis
#Title
RNA-seq Expression Analysis of Abiotic Stress in Arabidopsis thaliana

#Overview
This project focuses on analyzing RNA-seq data to study gene expression patterns under abiotic stress conditions in Arabidopsis thaliana.

#Objectives
- Perform basic RNA-seq data analysis
- Identify gene expression patterns
- Visualize data using PCA and heatmaps

#Tools Used
- Python (pandas, numpy, matplotlib, seaborn)
- Jupyter Notebook

#Workflow
1. Data collection
2. Data preprocessing
3. PCA analysis
4. Heatmap visualization

#Results
- PCA analysis shows separation between control and stress samples
- Heatmap highlights top variable genes across samples

#Skills Demonstrated
- RNA-seq data handling
- Data normalization and scaling
- PCA (Principal Component Analysis)
- Heatmap visualization
